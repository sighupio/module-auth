# Auth Module Release v0.7.0

Welcome to the latest release of the Auth module for the SIGHUP Distribution.

This release updates Pomerium and particularly includes an update to Envoy v1.35.6, to address the following CVEs in Envoy and its dependencies:

- CVE-2025-62504
- CVE-2025-62409
- CVE-2025-27818
- CVE-2025-27817
- CVE-2025-0913
- CVE-2024-25178
- CVE-2024-25177
- CVE-2024-25176

## Compatibility

This release adds support for Kubernetes 1.34.x while maintaining compatibility with previous versions (1.31.x - 1.33.x).

### Process

To upgrade this module from `v0.6.0` to `v0.7.0`, you need to download this new version, then apply the `kustomize` project. No breaking changes or configuration updates are required.

```bash
kustomize build | kubectl apply -f -
```
